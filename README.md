# SystemCOO / ATLAS

Голосова та візуальна операційна система (ATLAS).

## Запуск

- **GUI:** `python main_with_atlas.py`
- **Telegram-міст:** `python run_telegram_bridge.py`

Конфігурація: `config.py`. Голосовий вивід: `core/voice_output.py`; зір і жести: `core/vision_manager.py`, `core/visual_bridge.py`.
